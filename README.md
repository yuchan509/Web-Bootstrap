# Web-Bootstrap
Bootstrap templates 활용한 web 구성

### Python Web
- 목적 : 파이썬으로 웹 프로그램 구현.
- why  : 파이널 프로젝트에서 산출물(웹 + IOT 등 연결,(모바일,PC,장비), GUI+IOT 등 조합이 가능, 레포트)
- 과목 : 
  - 클라이언트 사이드 => 데이터 수집시 클라인언트를 분석해서 데이터를 수집하는데 베이스 지식.
    - html, css, javascript
  - 서버 사이드
    - Python 기반(Flask or django)
  - 데이터 베이스
    - mysql 계열(Maria db)
